# EduGuide AI - Coaching Institute Assistant

## Overview

EduGuide AI is a web-based chatbot and landing page for a coaching institute. It helps students explore courses, fees, admissions, and institute information.

## Features

- Modern coaching institute landing page
- AI chatbot for student queries
- Restricted responses only to institute-related questions
- Course recommendation support
- Admission and fee assistance

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)

## Run Project

1. Install dependencies:
   pip install -r requirements.txt
2. Run Flask app:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
